Projet | JavaScript - Exercices basiques
--------------------------------------------

Repository contenant le projet du 34ème jour de The Hacking Project

Ce projet contiendra essentiellement des déclarations de variables, boucles et autres joyeusetés présentées dans la ressource du jour. Le but est de maitriser les bases avant d'apprendre comment animer son site en JS.

Pour tester les scripts des différents exercices, il faut les appeler dans le index.html et ouvrir l'index dans son navigateur. Les résultats seront visibles dans la console (accessible avec F12).
